Database Views 

1. Active Courses View: 
o Create a view to display all active courses, including category, instructor name, and total 
enrollments. 

SELECT Courses.Title, Categories.CategoryName, Users.Email AS InstructorName, COUNT(Enrollments.EnrollmentID) AS TotalEnrollments
FROM Courses
JOIN Categories ON Courses.CategoryID = Categories.CategoryID
JOIN Users ON Courses.InstructorID = Users.UserID
LEFT JOIN Enrollments ON Courses.CourseID = Enrollments.CourseID
GROUP BY Courses.CourseID, Courses.Title, Categories.CategoryName, Users.Email;


2. Learner Progress View: 
o Implement a view to show progress for all learners, including course details and 
completion percentages.

SELECT Users.Email AS LearnerEmail, Courses.Title AS CourseTitle, 
       (COUNT(CASE WHEN Enrollments.Progress >= Lessons.LessonID THEN 1 END) * 100 / COUNT(Lessons.LessonID)) AS CompletionPercentage
FROM Enrollments
JOIN Users ON Enrollments.LearnerID = Users.UserID
JOIN Courses ON Enrollments.CourseID = Courses.CourseID
JOIN Lessons ON Courses.CourseID = Lessons.CourseID
GROUP BY Users.Email, Courses.Title;


3. Revenue Summary View: 
o Create a view summarizing revenue generated by each course, instructor, and overall 
platform.


SELECT Courses.Title AS CourseTitle, Users.Email AS InstructorEmail, SUM(Revenue.Amount) AS TotalRevenue
FROM Revenue
JOIN Courses ON Revenue.CourseID = Courses.CourseID
JOIN Users ON Revenue.InstructorID = Users.UserID
GROUP BY Courses.Title, Users.Email;
